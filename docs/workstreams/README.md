# Workstreams

Placeholder for future Beads-backed workstreams.

Expected structure once bootstrapped:

```text
docs/workstreams/
  status.md                  # generated from Beads
  ideas.md                   # generated from Beads
  backlog.md                 # generated from Beads
  <name>/
    README.md                # hand-authored charter
    roadmap.md               # hand-authored phased plan
    plans/                   # hand-authored per-phase plans
    tracking/                # generated from Beads
```

Generated mirror files must be produced from Beads, not hand-edited. Until the renderer exists, keep
this folder as a placeholder and do not invent status files.
