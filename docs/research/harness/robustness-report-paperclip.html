5 · Per-tool verdict

codegraph — best all-round semantic tool (F1 0.85 with real precision). Resolves barrels + cross-package, captures JSX and hooks (contradicting its capability card), indexes secrets.ts. Only weakness: heavy same-name collisions (name-keyed).
serena — best collision/disambiguation (LSP, collision-P 1.00) & highest precision (0.94), but barrel-blind (stops at the barrel index — shared with UA), lowest recall (0.59), cold-cache flaky (first query 0/1, warm correct). Highest setup cost (MCP daemon). Secrets visible.
codebase-memory-mcp — two catastrophic reference-style-dependent blind spots: db-table references (A8→0, A7→14/123) and forwardRef components (A18 invisible; A17 only 5/25 — so it does not "win" JSX). Wins the createApp collision (A2 1.00, file-scoped). Git-bleed confirmed (reports parent repo HEAD). Indexes secrets.
graphify — strong on db-schema/type/barrel via real re_exports edges; clean package dep graph (1 cycle: server↔scripts). Polymorphic-dispatch blind (A16 0.00). Silently drops 34 secret files (entire server/src/secrets/, Secrets.tsx, company_secrets table, .env.example) with no warning.
ast-grep / repomix — honest lexical baselines. Recall 1.0, precision collapses on collisions (A2 0.02), output up to 35× the answer. Stateless = always fresh. repomix strengths: comments survive (S12), native context-pack (S16: 171,608 tokens for a 3-file slice), indexes secrets.
understand-anything — weakest at symbol nav (barrel-blind on A9/A10/A11/A14, interfaces invisible, JSX uncaptured) but good at coarse architecture/hubs/layers. File/module-level graph, no symbol binding.

6 · Honesty & isolation

Abstention (corrected after Codex review): the 7 designed dimension-probes (rationale/runtime/ data-flow/async/migration/history/failure-mode) were deferred, not run. What held: static tools return file:line-or-nothing (cannot fabricate); workers reported per-scenario support honestly. The one safety failure is a tool — graphify's silent secret-file drop.
Isolation audit: subject unmutated (read-only), no global installs, no result bleed, outputs contained. One low-severity leak — codegraph telemetry queue escaped to ~/.codegraph (no subject content, offline, results valid; kill-switch fix scheduled for R2/R3).
Ground truth: harvest2.py (binding-resolved, transitive re-export closure, 38-package dynamic map), Codex gpt-5.5 xhigh cross-checked (caught + fixed the A16 mixed-import undercount).
