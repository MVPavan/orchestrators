Operations Checklist

Before runtime claims

  • Install dependencies in external/paperclip.
  • Run pnpm typecheck and pnpm test.
  • Boot pnpm dev and verify DB, migrations, health route, UI, and scheduler logs.

Before production claims

  • Verify external Postgres migration path.
  • Audit authenticated deployment exposure and trusted origins.
  • Check backup schedule and restore story.

Before plugin trust

  • Review manifest validation and capability gates.
  • Check plugin database namespace isolation.
  • Review worker environment and host handlers.

Before adapter trust

  • Inspect workspace path and remote target handling.
  • Check JWT/token scoping.
  • Confirm log redaction and command execution behavior.
