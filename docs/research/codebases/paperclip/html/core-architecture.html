Boundary Decisions

Server is the source of truth

The UI, CLI, and MCP package operate as clients. The server owns business rules, durable state transitions, plugin loading, adapter registration, schedulers, recovery, and live events.

Adapters execute, plugins extend

Adapters are the concrete agent execution boundary. Plugins are broader extension modules that can add workers, tools, jobs, webhooks, UI, and external adapters.

Workspaces isolate work

The heartbeat path creates or reuses execution workspaces and then acquires environment leases before adapter execution.

Database-backed operations

Issues, wakeups, heartbeat runs, runtime services, costs, activity, plugin jobs, and workspaces all have durable state.
