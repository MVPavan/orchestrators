Security-Sensitive Edges

Adapters

Adapters receive execution context, workspace/transport details, log callbacks, spawn hooks, and optionally local agent JWTs. Treat new adapters as privileged execution code.

Plugins

Plugins can add jobs, tools, webhooks, migrations, and workers. Manifest validation and capability-gated host handlers are critical.

Auth

Local trusted board mode and authenticated Better Auth/API key/JWT paths have different threat models. Source: server/src/middleware/auth.ts:22-280.

Providers

Storage and secrets providers must preserve company isolation, traversal protection, redaction, and runtime binding behavior.
