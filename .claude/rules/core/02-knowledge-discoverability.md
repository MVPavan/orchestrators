# Knowledge Discoverability

- If the repo has durable docs or learnings, make them discoverable from `AGENTS.md`, `CLAUDE.md`, or `.claude/project/docs-index.md`.
- An agent should be able to tell what exists, which doc is authoritative, and when it is worth reading.
- Prefer one small discoverability note over duplicated explanations.
- Keep long-lived docs repo-relative and stable.
