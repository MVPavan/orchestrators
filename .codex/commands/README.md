# Codex Commands

Codex custom prompts are deprecated for shared repo workflows. The Claude slash commands were migrated into Codex skills instead:

- `$adopt`
- `$check-invariants`
- `$prepare-phases`
- `$run-phases`
- `$use-codex`

Keep this directory as a migration note. Add repo-shared reusable workflows under `.codex/skills/`.
