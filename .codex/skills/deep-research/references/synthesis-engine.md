# Synthesis Engine

Run this after source collection and before writing the HTML artifact.

## 1. Claim Table

List the 5-10 claims that matter most.

For each claim record:
- Evidence.
- Source type.
- Date.
- Opposing evidence.
- Confidence.
- Why it matters.

## 2. Pattern Scan

Look for:
- Convergence: unrelated sources point to the same conclusion.
- Divergence: different segments, time periods, or incentives produce different answers.
- Silence: expected evidence is missing.
- Trend: direction changes over time.
- Bottleneck: one constraint determines whether the rest matters.

## 3. Fact To Insight

Turn facts into implications.

Formula:
`fact + context + implication for the user's decision`

If a paragraph does not answer "so what?", cut it or rewrite it.

## 4. Red Team

Before writing the conclusion, answer:
- What is the strongest case against this conclusion?
- What would make the opposite conclusion true?
- Are we seeing only winners, recent data, or vendor-friendly evidence?
- What source incentives could distort the picture?
- What failure mode would make this recommendation expensive or wrong?

Include the strongest counterargument in the artifact.

## 5. Confidence Calibration

Use:
- High: strong primary/current evidence and independent agreement.
- Medium: credible but incomplete evidence.
- Low: narrow, anecdotal, stale, or conflicting evidence.
- Speculative: reasoned inference; label assumptions.

## 6. Final Narrative

Write in this order unless the topic demands otherwise:
1. Answer.
2. Scope and method.
3. Landscape or options.
4. Evidence and tradeoffs.
5. Counterarguments.
6. Decision points or recommendation.
7. Open questions and next research.
